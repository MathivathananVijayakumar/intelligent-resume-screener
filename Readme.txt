This is a simple project using Streamlit and OpenAI API to intelligently analyze and match Resumes against Job Descriptions (JD).

## Features
- Extracts key skills from Resume and JD.
- Calculates a Match Score (%).
- Suggests missing skills.
- Provides professional feedback.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Set your OpenAI API key in `.env`
3. Run: `streamlit run app.py`
